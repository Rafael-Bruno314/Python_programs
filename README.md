# Python Programs

Repositório com programas e scripts em Python desenvolvidos por Rafael Bruno. Contém utilitários para manipulação de arquivos (🔧), registros de cursos (📚) e projetos educacionais (📖).

## 📂 Descrição das Pastas e Scripts

- [🔧 Baixar vídeo YouTube com múltiplas linhas de áudio](https://github.com/.../Baixar%20video%20youtube%20com%20multiplas%20linhas%20de%20audio)  
  Download de vídeos e playlists do YouTube com diferentes faixas de áudio.

- [🔧 Compressão de PDFs](https://github.com/.../Compress%C3%A3o%20de%20PDFs)  
  Redução do tamanho de arquivos .pdf em lote, com preservação considerável da qualidade.

- [🔧 Compressão de imagens](https://github.com/.../Compress%C3%A3o%20de%20imagens)  
  Redução automática do tamanho de imagens (JPEG/PNG) em lote, basta escolher a pasta contendo as imagens e o programa se encarrega de todas.

- [🔧 Compressão de vídeo](https://github.com/.../Compress%C3%A3o%20de%20v%C3%ADdeo)  
  Ferramenta para compressão de vídeos, com parâmetros ajustáveis pelo usuário.

- [🔧 Converter PDF em Word](https://github.com/.../Converter%20PDF%20em%20Word)  
  Conversão de PDFs para arquivos .docx editáveis.

- [📖 Gerenciador de Biblioteca](https://github.com/.../Gerenciador%20de%20Biblioteca)  
  Um pequeno sistema para catalogar livros, permitindo adicionar, listar e remover itens do acervo, tendo como diferencial um banco de dados em .txt.

- [📖 Método de Monte Carlo](https://github.com/.../M%C3%A9todo%20de%20Monte%20Carlo)  
  Pequenas simulações estatísticas usando o método Monte Carlo.

- [📚 Práticas do full course](https://github.com/.../Pr%C3%A1ticas%20do%20full%20course)  
  Conjunto de exercícios e exemplos práticos desenvolvidos durante um [curso completo de Python](https://www.youtube.com/watch?v=WGJJIrtnfpk). Inclui manipulação de listas, funções, classes, entre outros.

- [🔧 Reordenar PDFs](https://github.com/.../Reordenar%20PDFs)  
  Reorganização personalizada de páginas em documentos PDF.

- [📚 Semana do Python](https://github.com/.../Semana%20do%20Python)  
  Scripts desenvolvidos durante a participação em um mini curso de Python, focando em automações, processamento de dados e práticas diversas.

- [📖 Trabalhando com Numpy](https://github.com/.../Trabalhando%20com%20Numpy)  
  Exercícios sobre operações em arrays, estatística básica e manipulações matemáticas com NumPy.

- [📖 Unindo informações de revistas do Qualis](https://github.com/.../Unindo%20infos%20de%20revistas%20do%20Qualis)  
  Automação da organização de dados de revistas científicas Qualis, permitindo identificar todas as de maior interesse para o usuário (com base nas áreas e classificações das revistas).

- [🔧 Atualização em massa de pacotes Python](https://github.com/.../Atualiza%C3%A7%C3%A3o%20em%20massa%20de%20todos%20os%20pacotes%20python.py)  
  Script para atualizar automaticamente todos os pacotes instalados no ambiente Python para as últimas versões disponíveis.
